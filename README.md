# 💰 Sistema Bancário em Java

Projeto simples desenvolvido em Java para simular operações bancárias básicas.

## 🚀 Funcionalidades
- Criar conta
- Depositar valor
- Sacar valor (com limite)
- Exibir dados da conta

## 🛠 Tecnologias
- Java
- Programação Orientada a Objetos (POO)

## 📚 Conceitos aplicados
- Encapsulamento
- Métodos
- Construtores
- Regras de negócio
